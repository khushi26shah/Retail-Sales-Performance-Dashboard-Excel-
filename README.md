# Retail-Sales-Performance-Dashboard-Excel-
📊 Project Overview

This project presents an interactive Retail Sales Performance Dashboard built using Microsoft Excel. The dashboard analyzes sales data to uncover trends, customer behavior, regional performance, and order status.

It helps businesses quickly understand which products sell the most, which states generate the highest revenue, and how customers interact with the store across different sales channels.

The goal of this project is to demonstrate data cleaning, data analysis, and dashboard creation skills using Excel.

🎯 Key Insights

Women customers contribute the majority of sales (~69%), indicating a strong female customer base.

Adult women (20–40 years) represent the highest purchasing segment.

Top performing states include Maharashtra, Karnataka, and Uttar Pradesh.

93% of orders are successfully delivered, showing efficient order fulfillment.

Sales are distributed across multiple e-commerce platforms like Amazon, Flipkart, Myntra, and Meesho.

📈 Dashboard Features

The dashboard includes the following visual insights:

Monthly Order vs Sales Analysis
Compare total orders and revenue trends across months.

Customer Demographics Analysis
Sales breakdown by gender and age groups.

State-wise Sales Performance
Identify top-performing regions contributing to revenue.

Order Status Overview
Track delivered, returned, cancelled, and refunded orders.

Sales Channel Distribution
Analyze sales contribution from different e-commerce platforms.

Product Category Performance
Identify which categories drive the most sales.
