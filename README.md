# 🛍️ Retail Sales Performance Dashboard (Excel)
Excel Data Analysis Project | Sales Dashboard | Data Visualization

![Retail Sales Dashboard](retail-sales-dashboard.png)


## 📊 Project Overview
This project presents an interactive **Retail Sales Performance Dashboard** created using **Microsoft Excel**. The dashboard analyzes retail sales data to identify trends, customer behavior, and regional performance.

It helps transform raw sales data into meaningful insights through visualizations and KPIs, allowing users to quickly understand business performance.

This project demonstrates skills in **data cleaning, data analysis, and dashboard creation using Excel**.

---

## 🎯 Key Insights

- Women customers contribute around **69% of total sales**.
- **Adult women (20–40 years)** represent the largest customer segment.
- **Top performing states** include Maharashtra, Karnataka, and Uttar Pradesh.
- Around **93% of orders are successfully delivered**.
- Sales are distributed across multiple platforms such as **Amazon, Flipkart, Myntra, and Meesho**.

---

## 📈 Dashboard Features

The dashboard provides the following insights:

- **Order vs Sales Analysis**  
  Monthly comparison of order volume and revenue trends.

- **Customer Demographics**  
  Sales distribution based on gender and age groups.

- **State-wise Sales Performance**  
  Identification of top-performing states.

- **Order Status Analysis**  
  Breakdown of delivered, returned, cancelled, and refunded orders.

- **Sales Channel Analysis**  
  Performance comparison across different e-commerce platforms.

- **Product Category Analysis**  
  Understanding which product categories generate the most sales.
